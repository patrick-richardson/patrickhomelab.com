Core Infrastructure Systems

Forge

Primary Proxmox hypervisor hosting the virtual infrastructure including monitoring, automation, networking, and media services.

Vault

Dedicated storage node providing NFS storage used for media libraries, system backups, and persistent data for infrastructure services.

Controller-01

Ansible automation control node responsible for configuration management, patch automation, and infrastructure orchestration.

Monitor-01

Prometheus and Grafana monitoring stack used to observe system health, performance metrics, and infrastructure telemetry.

VPN Gateway

WireGuard VPN server providing secure remote connectivity into the PatrickHomeLab network environment.

Proxy-01

Reverse proxy responsible for internal service routing and exposing lab services through secure entry points.

MacBook Operations Interface

Primary workstation used to manage PatrickHomeLab infrastructure through SSH, Proxmox web UI, Ansible automation, and monitoring dashboards.
